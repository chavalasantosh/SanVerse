# soma_core_core/ - Structure Metrics System

## Overview

SOMA Core structure metrics system for pattern analysis, symbol structures, and hierarchical processing.

## Files

| File                     | Description                       |
| ------------------------ | --------------------------------- |
| `structure_hierarchy.py` | Hierarchical structure processing |
| `pattern_builder.py`     | Pattern building and analysis     |
| `symbol_structures.py`   | Symbol structure definitions      |
| `simple_metrics.py`      | Simple metrics calculations       |
| `embeddings.py`          | Structure embeddings              |
| `similarity.py`          | Similarity calculations           |
| `tokenizer.py`           | Structure-aware tokenizer         |
| `token_record.py`        | Token record management           |
| `train_embeddings.py`    | Embeddings training               |

## Key Concepts

- **Structure Hierarchy**: Hierarchical representation of text structures
- **Pattern Building**: Pattern recognition and building
- **Symbol Structures**: Mathematical symbol representations
