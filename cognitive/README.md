# cognitive/ - AI/ML & Cognitive Systems

## Overview

Advanced AI/ML capabilities including reasoning, SLM models, graph processing, and cognitive pipelines.

## Modules

| Module         | Description                                                       |
| -------------- | ----------------------------------------------------------------- |
| `slm/`         | Small Language Models - santok_gpt, tiny_slm, slm_trainer         |
| `reasoning/`   | Reasoning engine - inference_engine, hybrid_reasoner, explainer   |
| `algorithms/`  | Algorithms - santek_metrics, pattern_matcher, semantic_similarity |
| `graph/`       | Graph processing - graph_store, graph_node, graph_edge            |
| `trees/`       | Tree structures - tree, tree_node, tree_store                     |
| `memory/`      | Memory systems - unified_memory, memory_object                    |
| `integration/` | Cognitive pipelines - embedding_bridge, token_bridge              |
| `utils/`       | Utilities - formatting, scoring, validation                       |

## Key Files

- `slm/santok_gpt.py` - SanTOK GPT model
- `slm/tiny_slm.py` - Tiny Small Language Model
- `reasoning/inference_engine.py` - Inference engine
- `algorithms/santek_metrics.py` - SanTEK metrics system
