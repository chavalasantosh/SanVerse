# semantic_trainer/ - Enhanced Semantic Embeddings

## Overview

Enhanced semantic trainer for generating high-quality semantic embeddings.

## Files

| File                  | Description                    |
| --------------------- | ------------------------------ |
| `enhanced_trainer.py` | Main enhanced semantic trainer |
| `example_train.py`    | Training example               |
| `example_use.py`      | Usage example                  |
| `examples/`           | Additional examples            |

## Usage

```python
from semantic_trainer import EnhancedSemanticTrainer

trainer = EnhancedSemanticTrainer()
trainer.train(corpus_path="data/corpus.txt")
```
