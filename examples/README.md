# examples/ - Usage Examples

## Overview

Usage examples demonstrating SanTOK capabilities.

## Examples

| File                                    | Description                     |
| --------------------------------------- | ------------------------------- |
| `embedding_example.py`                  | Embedding generation example    |
| `search_examples.py`                    | Search functionality examples   |
| `train_semantic_embeddings.py`          | Semantic embeddings training    |
| `use_vector_store.py`                   | Vector store usage              |
| `comprehensive_vector_store_example.py` | Comprehensive vector store demo |
| `integration_with_transformers.py`      | Transformers integration        |
| `train_santok_60k_vocab.py`             | 60K vocabulary training         |

## Running Examples

```bash
python examples/embedding_example.py
python examples/search_examples.py
```
